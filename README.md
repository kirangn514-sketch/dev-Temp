## 📖 Topic Summaries

### 🌐 [HTML & CSS](./HTML_&_CSS.md)
- Semantic HTML5 elements and accessibility
- CSS Box Model, positioning, and specificity
- Flexbox and CSS Grid layouts
- Responsive design and media queries
- CSS variables, animations, and transitions

---

### ⚡ [JavaScript — Core](./JS.md)
- Data types, type coercion, and typeof
- Functions, scope, and hoisting
- Arrays, objects, and built-in methods
- DOM manipulation and events
- Error handling with try/catch

---

### 🔥 [JavaScript — Advanced](./Javascript.md)
- ES6+ features: destructuring, spread, optional chaining
- Closures, prototypes, and inheritance
- Promises, async/await, and the event loop
- Modules (CommonJS vs ES Modules)
- Memory management and performance

---

### ⚛️ [React.js — Concepts](./ReactJs.md)
- JSX, components, and props
- useState, useEffect, useRef, useCallback, useMemo
- Virtual DOM and reconciliation
- Controlled vs uncontrolled components
- Component lifecycle

---

### ⚛️ [React.js — Deep Dive](./react.md)
- Performance optimization (React.memo, lazy loading)
- State management: Redux, Context API
- Custom hooks and patterns
- Code splitting and Suspense
- Testing React components

---

### 🟢 [Node.js](./node.md)
- Node.js architecture and event loop
- Building REST APIs with Express.js
- Middleware, routing, and error handling
- Authentication with JWT
- Working with databases from Node.js

---

### 🗄️ [SQL — Concepts](./SQL.md)
- DDL, DML, DQL, DCL commands
- Joins: INNER, LEFT, RIGHT, FULL, SELF, CROSS
- Indexes, Views, Stored Procedures, Triggers
- Transactions and ACID properties
- Window functions and CTEs
- Normalization: 1NF, 2NF, 3NF

---

### ❓ [SQL — Interview Q&A](./SQL_QA.md)
- 70+ interview questions with detailed answers
- Common tricky questions (NULL traps, NOT IN, etc.)
- Query patterns: duplicates, top-N, running totals
- Optimization tips and execution order
- Real-world SQL scenarios

---

### 🔷 [TypeScript](./Typescript.md)
- Type annotations and inference
- Interfaces vs Types
- Generics and utility types
- Enums and decorators
- Strict mode and compiler options
- TypeScript with React

---

### 🏗️ [System Design](./SystemDesign.md)
- Scalability and load balancing
- Caching strategies (Redis, CDN)
- Database design and sharding
- REST vs GraphQL API design
- Microservices architecture
- Real-world design problems
